# A1 Deutsch lernen

A modern, responsive learning platform for building practical German vocabulary at the A1 level.

## About the platform

The platform turns themed A1 vocabulary lists into interactive learning experiences. Learners choose a topic before starting, so only the selected learning content is loaded.

The dark interface is designed for desktop computers, tablets, and mobile phones.

## Available topics

### Menschen und persönliche Wörter

Vocabulary related to:

- personal information;
- family and people;
- countries and languages;
- professions;
- feelings;
- hobbies.

### Haus und Wohnen

Vocabulary related to:

- rooms and furniture;
- household objects;
- kitchen and bathroom items;
- apartments and buildings;
- household activities;
- room descriptions.

## Learning activities

- Interactive vocabulary flashcards
- German pronunciation
- Ukrainian translations
- German example sentences
- Color-coded noun articles
- Advanced written vocabulary tests
- Listening comprehension tests
- Sentence completion exercises
- Grammar exercises
- Sentence-building activities

## Learning progress

The platform keeps learning progress in the browser and includes:

- learned-word tracking;
- consecutive-day learning streaks;
- active study-time tracking;
- a daily learning plan;
- task completion indicators;
- a personal mistake table;
- automatic removal of corrected mistakes.

## Experience

- A modern visual style with a dedicated color theme for each topic
- A comfortable dark interface throughout the platform
- Randomized exercises on every attempt
- Responsive layouts for phones, tablets, and desktops
- Topic-based loading for faster and more focused sessions
